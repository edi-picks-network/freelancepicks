# freelancepicks-hub

**FreelancePicks Studio** — a freelance platform strategy site with a warm green / nature-inspired color scheme.

Built with Next.js 15, Tailwind CSS v3.4, and TypeScript.

Color palette (Warm Green / Nature):
- Primary bg: #F5F9F0 (warm-bg)
- Card bg: #FCFDF9 (warm-card)
- Green primary: #2D8A4E (leaf)
- Green light: #4CAF50 (leaf-light)
- Green lighter: #81C784 (leaf-lighter)
- Green subtle: #E8F5E9 (leaf-subtle)
- Green dark: #1B5E20 (leaf-dark)
- Border soft: #DCE8D5 (border-soft)
- Text primary: #1A2E1D (text-primary)
- Text secondary: #4A6B4F (text-secondary)
- Text muted: #7A9A7F (text-muted)
- Accent earth: #8D6E63 (earth)
- Accent gold: #D4A843 (earth-gold)
