# freelancehubs

**FreelancePicks** — a freelance platform directory and review site with a violet color scheme.

Built with Next.js 15, Tailwind CSS v3.4, and TypeScript.

Color palette (Violet):
- Primary bg: #0A0A1A (deep black)
- Secondary: #111128 (dark)
- Tertiary: #1A1A3E
- Border: #2D2D6B
- Active border: #8B5CF6 (violet-500)
- Accent: #8B5CF6 (violet-500)
- Accent glow: #A78BFA (violet-400)
- Text primary: #F0F0FF (near-white)
- Text secondary: #C4B5FD
- Text muted: #6B7280 (gray-500)
- Gold/Star: #F59E0B
